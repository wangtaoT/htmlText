# htmlText
### 支持标签

- `<p>`
- `<div>`
- `<br>`
- `<b>`
- `<i>`
- `<strong>`
- `<em>`
- `<u>`
- `<tt>`
- `<dfn>`
- `<sub>`
- `<sup>`
- `<blockquote>`
- `<cite>`
- `<big>`
- `<small>`
- `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`
- `<a href="...">`
- `<ul>`
- `<ol>`
- `<li>`
- `<code>`
- `<center>`
- `<strike>`
- `<div>`
- `<font size="..." color="...">`
- `< img src="..." width="..." height="...">`
- `<span style="font-size: 14px; color: rgb(230, 0, 0);">...</span>`
